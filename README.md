# Gomoku Professional Edition

A web-based professional version of the Gomoku (Five in a Row) game, featuring enhanced rules and gameplay mechanics.

## Features

### 🎮 Gameplay
- **Enhanced Rules**: Implements a professional rule where Black is prohibited from forming two lines of five simultaneously.
- **Dynamic Board Size**: Customize the board dimensions.
- **Undo System**: Revert moves with a configurable limit.
- **Match History**: Real-time log of all moves, including victories, draws, and undos.

### 🤖 AI & Timer
- **Countdown Timer**: Turn-based timer to keep the game pace.
- **Automated Play**: When the timer expires, an AI opponent makes a move (proximity-based logic).
- **Toggleable Timer**: Option to enable or disable the countdown and auto-play.

### 🎨 UI/UX
- **Visual Indicators**: Current turn is highlighted with a piece icon.
- **Canvas Rendering**: Smooth, wood-themed board graphics using HTML5 Canvas.
- **Victory Effects**: Visual animations for winning lines and board dimming upon game completion.
- **Responsive Controls**: Easy-to-use interface for restarting and undoing moves.

### 🛠️ Technical Stack
- **HTML5**: Structure and Canvas API.
- **CSS3**: Modern styling with CSS Variables.
- **JavaScript (ES6+)**: Object-oriented game logic.
